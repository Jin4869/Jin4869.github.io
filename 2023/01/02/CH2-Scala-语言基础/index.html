2.1 Scala 语言概述

Scala 是一门纯粹的面向对象的语言。每个值都是对象，每个操作都是方法调用。对象的数据类型以及行为由类和特质描述。

也是一门函数式语言。每个函数都是一个值，并且和其他类型（如整数、字符串等）的值处于同一地位。

2.2.1 基本数据类型和变量

基本数据类型

和 Java 不同的是，在 Scala 中，这些类型都是“类”，并且都是包 scala 的成员，比如，Int 的全名是 scala.Int。对于字符串，Scala 用 java.lang.String 类来表示字符串

字面量

操作符
优先级：算数运算符 > 关系运算符 > 逻辑运算符 > 赋值运算符

  • 算术运算符：加（+）、减（-）、乘（*）、除（/）、余数（%）；
  • 关系运算符：大于（>）、小于(<) 、等于（==）不等于（!=）、大于等于（>=）、小于等于(<=)
  • 逻辑运算符：逻辑与(&&)、逻辑或(||)、逻辑非(!)；
  • 位运算符：按位与(＆)、按位或(|)、按位异或(^)、按位取反(~)等
  • 赋值运算符：=及其与其它运算符结合的扩展赋值运算符，例如+=、
    %

变量

  • val：不可变的，声明时必须被初始化，初始化后不能再赋值
  • var：可变的，声明时需要进行初始化，初始化后还可以再次赋值

根据初始值可以省略具体的数据类型

也可以显式声明变量的类型

2.2.4.3 容器（Collection）

包括序列（Sequence）、集合（Set）、映射（Map）

Scala 用了三个包来组织容器类，分别是 scala.collection, scala.collection.mutable, scala.collection.immutable

scala.collection 定义了可变容器和不可变容器的一些通用操作

2.2.4.4 序列（Sequence）

元素可以按照特定的顺序访问的容器，序列中每个元素均带有一个从 0 开始计数的固定索引位置

具有两个子特质 LinearSeq 和 IndexedSeq，LinearSeq 序列具有高效的 head 和 tail 操作，而 IndexedSeq 序列具有高效的随机存储操作

列表（List）

scala 中的 List 一旦被定义，其值就不能改变

构造列表

除了 head、tail 操作是常数时间 O(1)，其他按索引访问的操作都需要从头开始遍历，即 O(N)

向量（Vector）
实现所有访问操作都是常数时间

Vector 不能改变，但是可以通过生成新的 Vector 来实现对 Vector 的修改

++ 连接两个集合
::: 连接两个 List 类型的集合
这些操作符都是从右往左结合的

Range

一种特殊的、带索引的不可变数字等差序列

2.2.4.5 集合（Set）

不重复元素的容器

创建不可变集

创建可变集

操作：

  • head
  • tail
  • isEmpty，在集合为空时返回 true
  • 连接：++或.++()
  • 交集：.&()或.intersect()

2.2.4.6 映射（Map）

可变的映射

循环遍历映射

Map 合并：使用++或Map.++()，Map 合并时会移除重复的 key
如果存在相同的 key，合并之后的 Map 中的 value 会被最右边的 Map 的值所替代

2.2.4.7 迭代器（Iterator）

提供了按顺序访问容器元素的数据结构

操作：

  • next：返回迭代器的下一个元素
  • hasNext：检测是否还有下一个元素
  • .max查找最大元素，.min查找最小元素
  • .size,.length，获取迭代器的长度
